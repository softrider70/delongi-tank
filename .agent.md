# bosch-tank — Copilot Agent Configuration

This project uses GitHub Copilot for development assistance.

## Available Skills

Use these commands in Copilot chat:

```
/build              Build firmware, bump build number, commit build metadata
/build-project      Build firmware (alias for /build)
/flash              Flash device using the last used USB or OTA workflow
/upload-firmware    Flash app (~3 seconds)
/add-ota            Add OTA firmware updates
/add-webui          Add web dashboard
/add-library        Manage components
/add-security       Enable Secure Boot/encryption
/setup-ci           GitHub Actions CI/CD
/add-profiling      Performance monitoring
/update-inventory   Update desktop ESP hardware inventory from MAC/ESP scan
/commit             Git operations
```

## Project Context

- **Board:** Select from: esp32, esp32s2, esp32s3, esp32c3, esp32c6
- **Main code:** `components/main/main.c`
- **Config:** `include/config.h`
- **Build:** `CMakeLists.txt` + `sdkconfig*`

## Development Workflow

1. Ask Copilot questions about your code
2. Use `/build` or `/build-project` to compile, bump build number, and commit build metadata
3. Use `/flash` to flash the device with the last used USB/OTA workflow
4. Use `/upload-firmware` for fast app-only upload
5. Use `/add-*` skills to extend functionality
6. Use `/commit` to save changes
